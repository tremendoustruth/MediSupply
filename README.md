To run the application locally, open in IDE using terminal:
  
    git clone https://github.com/tremendoustruth/MediSupply.git  
* Install necessary packages
* cd into back-end/inventory
* Enter: node index.js
* Navigate to localhost:9876/api-docs to view swagger documentation
* cd into front-end
* Enter: npm run dev
* Navigate to localhost:5174 to browse products
